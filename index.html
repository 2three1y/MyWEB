About the Project

MasterDnsVPN is a specialized tool designed to optimize and secure your network traffic by combining the power of encrypted DNS, Virtual Private Network (VPN) routing, and Tor integration. It ensures that your DNS queries remain private while efficiently routing traffic through designated secure tunnels or the Onion network.

Architecture

The MasterDnsVPN architecture is built on four main pillars:

  • DNS Resolver: A local, high-performance DNS resolver that supports DoH (DNS over HTTPS) and DoT (DNS over TLS).
  • Routing Engine: Intelligent traffic analysis that determines which requests should bypass the VPN for speed and which require the tunnel for security.
  • VPN Integration: Seamless integration with standard protocols like WireGuard and OpenVPN.
  • Tor & Onion Routing: Privacy Focused Integrated support for .onion domains and optional multi-hop routing for sensitive traffic.

Get Started Flow

  1. Clone & Install:

    Pull the latest build from our GitHub Repository and install dependencies.

    git clone https://github.com/2three1y/master-dns-vpn-web.git
  2. Configure:

    Use the generator above or edit config.yaml manually to define your credentials.

  3. Launch:

    Start the service using Docker Compose.

    docker-compose up -d
