About the Project

MasterDnsVPN is a specialized tool designed to optimize and secure your network traffic by combining the power of encrypted DNS, Virtual Private Network (VPN) routing, and Tor integration. It ensures that your DNS queries remain private while efficiently routing traffic through designated secure tunnels or the Onion network.

Architecture

The MasterDnsVPN architecture is built on four main pillars:

  • DNS Resolver: High-performance local resolution with DoH/DoT capabilities.
  • Routing Engine: Intelligent traffic analysis for optimized tunnel routing.
  • VPN Integration: Native support for secure, modern tunneling protocols.
  • Tor & Onion Routing: Privacy Focused Integrated support for .onion domains.

Privacy & Tor Integration

Elevate your anonymity with built-in Tor support. MasterDnsVPN allows you to:

  • Access .onion Services: Resolve and route .onion addresses natively.
  • Traffic Masking: Route packets through the Tor network to hide origin IP.
  • Metadata Protection: Encapsulate DNS lookups within encrypted circuits.
