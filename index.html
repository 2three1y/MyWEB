MasterDnsVPN

Advanced Secure DNS-Based VPN Routing

About the Project

MasterDnsVPN is a specialized tool designed to optimize and secure your network traffic by combining the power of encrypted DNS with Virtual Private Network (VPN) routing. It ensures that your DNS queries remain private while efficiently routing traffic through designated secure tunnels.

Architecture

The MasterDnsVPN architecture is built on three main pillars:

  • DNS Resolver: A local, high-performance DNS resolver that supports DoH (DNS over HTTPS) and DoT (DNS over TLS).
  • Routing Engine: Intelligent traffic analysis that determines which requests should bypass the VPN for speed and which require the tunnel for security.
  • VPN Integration: Seamless integration with standard protocols like WireGuard and OpenVPN.

How It Works

  1. The user makes a network request.
  2. The DNS Resolver intercepts the query and fetches the IP via a secure channel.
  3. The Routing Engine checks the destination against user-defined rules.
  4. Traffic is encrypted and sent through the VPN Tunnel or the direct gateway.
